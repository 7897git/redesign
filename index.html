Dokumentasi API Pengembang

Integrasikan fungsionalitas desain ulang ini ke dalam aplikasi milikmu sendiri

GET /api/redesign

Query Parameters

Parameter Tipe Wajib Deskripsi
url String Ya URL target website yang ingin kamu desain ulang (pastikan di-encode).

Integrasi Contoh Code (JavaScript Fetch)

// Panggil API Redesigner secara asinkron dari frontend kamu
const target = encodeURIComponent("https://daffadevhosting.github.io");
const endpoint = "https://website-redesigner.warpzone.workers.dev/api/redesign?url=" + target;


fetch(endpoint)
  .then(res => res.text())
  .then(html => {
     console.log("Desain Baru Diterima!");
     document.getElementById("output").srcdoc = html;
  });
